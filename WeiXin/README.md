- 微信技巧
	- 客户咨询，怎样回复淘口令？
		- 字符串
			- 用户直接复制淘口令
			- 容易被微信识别-封号
			- 淘口令的识别码$￥，可以随机变化
		- 二维码
			- 用户常按二维码识别，再复制淘口令
		- 网址-H5
			- 用户打开后，再复制淘口令
			- 容易被封域名
				- 不停更换域名，斗智斗勇
	- 回复淘口令的频率不能太高，会引起封号
		- 解封技巧？
